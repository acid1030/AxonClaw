功能模块详解

从概览到系统，四大模块覆盖完整工作流

01

概览与对话系统

  • Dashboard：展示关键运行指标、任务态势与系统健康度
  • Chat：支持多会话消息交互、上下文延续与会话新建
  • Sessions：历史会话检索、切换与整理
02

配置中心

  • Models / Channels / Agents：模型、渠道与代理治理
  • Templates / Hooks：提示词模板与扩展处理链
  • Gateway / Session / Memory：运行参数与记忆规则
  • Logging / Commands / Misc：调试、审计与运维
03

代理与编排

  • Agents：代理资产管理、能力查看与状态维护
  • Workflow：可视化多代理协作流程编排
  • Nodes：节点资源管理与拓扑展示
  • Cron：定时调度与执行历史
04

技能、知识与运维

  • Skills：技能中心、插件扩展与市场接入
  • Knowledge：知识检索、分组浏览与统计
  • GatewayMonitoring：状态监控与事件分析
  • Settings / Usage：全局设置与用量统计

常见问题

关于 AxonClawX 的常见疑问解答

AxonClawX 是什么？

AxonClawX 是基于 Electron + React 19 的 OpenClaw 桌面客户端，提供多代理协作、配置中心、技能市场、知识中心、网关监控与系统管理能力。

需要编程基础吗？

不需要。AxonClawX 提供图形化界面，通过点击配置即可使用大部分功能。对于高级用户，也支持 CLI 模式和自定义扩展。

支持哪些平台？

AxonClawX 跨平台支持 macOS、Windows 和 Linux，基于 Electron 构建，确保在不同操作系统上的一致体验。

数据安全吗？

完全安全。所有数据本地存储，不经过任何第三方服务器。代码开源，可审计。API Key 加密存储。

免费吗？

是的，AxonClawX 完全开源免费，采用 MIT 协议。你可以自由使用、修改和分发。
